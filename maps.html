Geospatial Data Visualizations

This page showcases interactive maps and dashboards that communicate geospatial data effectively. Below is an example of a dashboard created on ArcGIS Online to visualize positive COVID-19 cases per MODZCTA (Modified Zip Code Tabulation Areas) in New York City.

COVID-19 Cases in NYC (MODZCTA)

Key Insights from the Dashboard

  • Hotspots: Areas with the highest concentration of positive cases.
  • Trends Over Time: Use the time slider to see how cases evolved during the pandemic.
  • Demographics: Explore how different neighborhoods were affected.
